HTML definitions to memeorize

  1. DOM-Document Object Model
    It's basically the entire webpage.
    Example: This webpage, since it is a webpage, it has a DOM.
  2. Node
    An object in the DOM
    Example: <p> tags, <body>, and <head> tags.
  3. Element
    The content between a start HTML tag and an end HTML tag.
    Example: <p>This is a paragraph.</p>
    "This is a paragraph" would be the element content for the paragraph tag.
